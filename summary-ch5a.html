1. ADS-B System

Automatic Dependent Surveillance-Broadcast (ADS-B) = ระบบที่รวม GPS + เครื่องส่ง/รับบนเครื่องบิน + ground stations เพื่อให้นักบินและ ATC ทราบตำแหน่งและความเร็วของเครื่องบินอย่างแม่นยำ (ส่วนหนึ่งของ FAA NextGen)

รูปแบบหน้าที่
ADS-B Outส่งสัญญาณ line of sight จากเครื่องบิน → ATC ground receivers และเครื่องบินอื่น
ADS-B Inต้องมีจึงจะรับสัญญาณและแสดง traffic ในเครื่องได้

บริการของ ADS-B

  • ให้ข้อมูล real-time แม่นยำ บอก ATC ทันทีเมื่อเครื่องเบี่ยงจาก flight path; ระยะ effective 100 nm
  • ADS-B In แสดง traffic บน CDTI (cockpit display of traffic information) — จอเฉพาะ หรือรวมใน GPS moving map / MFD
  • TIS-B (traffic information service-broadcast) — แสดงเครื่องที่มี transponder แต่ไม่มี ADS-B
  • ใช้ดู surface movement เพิ่ม situational awareness ขณะ taxi ลดความเสี่ยง runway incursion
  • FIS-B (flight information service-broadcast) — ส่งข้อมูลอากาศ (text+graphic), ตำแหน่ง TFR, สถานะ SUA

ADS-B Data Links

Data LinkระดับสูงTIS-BFIS-B
1090ES
(1090 MHz extended squitter)		ทุกระดับ; บังคับที่ ≥18,000 ft MSL✅❌
978 UAT
(978 MHz universal access transceiver)		เฉพาะ <18,000 ft MSL✅✅
💡 ต้องการบินทุกระดับ + รับ FIS-B: ส่ง (Out) บน 1090ES และรับ (In) ทั้ง 1090ES และ 978 UAT

2. Radar

ATC ใช้ radar คู่กับ ADS-B — เครื่องส่ง/รับวิทยุภาคพื้นที่ทำงาน sync กัน ใช้เสาอากาศหมุน เมื่อคลื่นกระทบเครื่องบินจะสะท้อนกลับมาที่เสา

ชนิดหลักการ
Primary Radarวัดเวลาที่ echo สะท้อนกลับ + ทิศทางของสัญญาณ → บอกระยะและ azimuth (ทิศ) ของเครื่องเทียบกับเสา
Secondary Surveillance Radar
(ATCRBS = air traffic control radar beacon system)		ATC กำหนด discrete code ที่นักบินใส่ใน transponder; interrogator ส่งสัญญาณ → transponder ตอบอัตโนมัติด้วยสัญญาณเฉพาะ (แรงกว่า primary มาก) → แสดง ID + altitude + ground speed; เสา secondary ติดบน primary แสดงรวมจอเดียว

ข้อจำกัดของ Radar (3 อย่าง — จุดสอบ)

ข้อจำกัดรายละเอียด
Bending
(anomalous propagation)		คลื่นถูกบิดจากปรากฏการณ์ เช่น temperature inversion — โค้งลงพื้น = เกิด ground clutter; โค้งขึ้น = ลดระยะตรวจจับ
Reflectedคลื่นสะท้อนจากวัตถุหนาแน่น (เมฆหนา, ฝน, สิ่งกีดขวาง, ภูเขา) → บังเครื่องที่ระยะเดียวกัน และลด/ลบ target ที่ไกลกว่า
Screenedภูมิประเทศสูงบัง → เครื่องระดับต่ำที่ถูกภูเขาบัง หรืออยู่ใต้คลื่นเพราะความโค้งโลก จะไม่ปรากฏ
กล่องเสริม: B-2 Bomber (เครื่องบินล่องหน)

B-2 ออกแบบให้มี radar return น้อยมาก — ผิวเคลือบ radar-absorbing + เรขาคณิตตัวเครื่องสะท้อนคลื่นออกห่างจากเครื่องส่ง + ช่องรับอากาศบังใบพัด ทำให้ radar cross-section เพียง ~0.1 ตร.ม.

3. Transponders

ทั้ง ADS-B และ radar ใช้ transponder ของเครื่องเพื่อให้ข้อมูลแม่นยำ (altitude, speed)

Transponder Modes

Modeทำอะไร
Mode Aส่งรหัส 4 หลักเพื่อระบุตัวตน (identification)
Mode Cส่ง altitude ให้ ATC
Mode Stwo-way data link, บังคับสำหรับ ADS-B; 1090ES Mode S ส่ง GPS data (ตำแหน่ง/ทิศ/ความเร็ว/vertical trend) ≥1 ครั้ง/วินาที; จำเป็นต่อ TCAS
Mode C / ADS-B Out บังคับใช้ที่: Class A, B, C · ภายใน 30 ไมล์จาก primary airport ของ Class B · เหนือ Class B และ C · เหนือ 10,000 ft MSL (ADS-B Out บังคับในพื้นที่เดียวกันตั้งแต่ปี 2020)

ปุ่มเลือกการทำงาน (Function Keys)

ปุ่มความหมาย
STBYไม่ตอบ interrogation
ONตอบ interrogation แต่ไม่มี altitude
ALTตอบ interrogation พร้อม altitude
OFFปิดเครื่อง
GNDตอบเฉพาะ Mode S interrogation (ไม่ตอบ Mode A/C)

Transponder Codes

Codeความหมาย
1200VFR ปกติ (ปุ่ม VFR หรือค่า default)
discreteATC กำหนด (Class A/B/C หรือเมื่อขอ traffic advisory)
7500Hijacking (ถูกจี้เครื่องบิน)
7600Communication failure (วิทยุเสีย)
7700Emergency (ไฟไหม้ / เครื่องยนต์ขัดข้อง ฯลฯ)
  • ตั้งได้ 4,096 codes (ใช้เลข 0–7)
  • 💡 ทิป: 7500 "taken alive" · 7600 "radio's nix" · 7700 "going to heaven"
ระวัง alarm: อย่าตั้งเลข 7 ในหน้าต่างแรกโดยไม่ตั้งใจ — เช่น เปลี่ยน 2700 → 7200 ให้ทำ 2700 → 2200 → 7200 (อย่าผ่าน 7700)

Squawk Phraseology

คำสั่ง ATCทำอะไร
"Squawk (number)"ตั้ง code ที่กำหนด
"Ident"กดปุ่ม IDENT
"Squawk VFR"ตั้ง 1200
"Squawk standby"สลับไป STBY
"Stop altitude squawk"ปิดการรายงาน altitude (มักเกิดเมื่อ altitude คลาดเกิน 300 ft)
"Stop squawk"สลับไป OFF
"Squawk MAYDAY"ตั้ง 7700
  • IDENT = ทำให้ return "blossom" (เด่นขึ้น) บนจอ ATC 2–3 วินาที ให้ controller ระบุตัวง่าย
  • ถ้าต้องสลับไป STBY/OFF → รอจนถึง parking
Transponder Inspection: FARs กำหนดให้ทดสอบ/ตรวจสอบทุก 24 calendar months สำหรับบินใน controlled airspace; ทำโดย certificated repair station และบันทึกใน aircraft logbook

4. Flight Service

ใช้ Flight Service ได้ทุกเฟส ตั้งแต่ preflight planning จนลงจอด

Preflight (วางแผน)

  • 1-800-WX-BRIEF หรือ 1800wxbrief.com: weather briefing, NOTAMs, flight plan aids (หา departure time / altitude / route ที่ดีที่สุด)
  • File flight plan (online/phone): ให้ข้อมูล destination, route, time enroute, จำนวนคนบนเครื่อง — VFR ไม่บังคับแต่แนะนำ เพื่อรับ search and rescue protection
  • ACAS (adverse condition alerting service): แจ้ง text/email หากมีสภาพไม่ดีเกิดขึ้นหลัง file

Departure → Enroute → Arrival

  • Activate: 122.2 MHz หรือ local FSS frequency หรือ EasyActivate (email ลิงก์ 30 นาทีก่อน ETD)
  • Enroute: briefer ให้อากาศอัปเดต/ช่วยเหลือถ้าหลงหรือฉุกเฉิน; นักบินควรให้ PIREPs + รายงาน ADS-B/TIS-B/FIS-B malfunction
  • Close: โทร หรือ EasyClose (email ลิงก์ 30 นาทีก่อน ETA อิงเวลาออกจริง)

5. Search and Rescue (SAR)

  • แจ้ง Flight Service ถ้า destination เปลี่ยน หรือจะช้ากว่าแผน ≥ 15 นาที
  • ถ้าไม่ close ภายใน 30 นาทีหลัง ETA → Flight Service เริ่มค้นหาทางโทรศัพท์ (destination, สนามตามเส้นทาง, home base, contacts ใน flight plan)
  • ถ้าโทรไม่สำเร็จ → alert ระบบ SAR (U.S. Coast Guard, Civil Air Patrol, law enforcement) เพื่อค้นหาและช่วยเหลือ
📡 Surveillance-enhanced SAR: ถ้าเครื่อง equipped จะส่ง position reports ระหว่างบิน — ระบบ alert Flight Service ทันทีถ้าเครื่องหยุดเคลื่อนหรือหยุดส่ง ทำให้ค้นหาในพื้นที่ถูกต้องได้เร็ว
กรณีศึกษา: CFIT — Hawker Jet ชน Otay Mountain (1991)

คืนเดือนมืดท้องฟ้าโล่ง เครื่อง Hawker (HS-125) ออกจาก Brown Field แบบ VFR เลือกบินใต้ Class B ของ San Diego เพื่อรอ IFR clearance — 2 นาทีหลัง takeoff ก่อนได้ clearance เครื่องชนภูเขาที่ 3,300 ft MSL ห่างจากยอดเขาแค่ 72 ฟุต เสียชีวิตทั้ง 10 คน บทเรียน: ใช้บริการ ATC ให้เต็มที่โดยเฉพาะพื้นที่ไม่คุ้น แต่อย่าประมาท — ศึกษาภูมิประเทศ + ใช้ chart/moving map/terrain awareness ป้องกัน Controlled Flight Into Terrain (CFIT)

6. Control Tower Services

หอบังคับการบินให้บริการในเขต terminal — ครอบคลุมตั้งแต่ก่อนบินจนลงจอด

ATIS (Automatic Terminal Information Service)

  • ฟังก่อนบินจากสนามที่มี tower เพื่อรับ airport advisory information
  • อัปเดตเมื่อ: airport conditions เปลี่ยน, มี official weather เข้ามา, และปกติทุกต้นชั่วโมง
  • label ด้วยตัวอักษร phonetic ต่อเนื่อง (Information Alpha, Bravo, ...); precorded, ออกอากาศต่อเนื่องบนความถี่เฉพาะ; สนามใหญ่อาจแยก ATIS สำหรับ departure/arrival
📻 ถ้า sky condition + visibility หายไปจาก ATIS = visibility ≥ 5 ไมล์ และ ceiling ≥ 5,000 ฟุต

ลำดับบริการ (Class D)

  1. Clearance Delivery (สนาม busy): ออก IFR clearances และอาจให้ VFR departure instructions + transponder code
  2. Ground Control: clearance taxi ใน movement areas (runways + taxiways ที่ไม่ใช่ ramp/parking)
  3. Tower: takeoff clearance; ให้ traffic advisories (เท่าที่ workload เอื้อ) + safety alerts
  4. ขาเข้า: ฟัง ATIS → tower สั่งเข้า traffic pattern + clear to land → Ground control นำ taxi ไป parking
💡 Progressive Taxi = ขอ controller บอกเส้นทาง taxi เป็นขั้นๆ (เมื่อไม่คุ้นสนาม)

7. TRACON & ARTCC Services

เครือข่าย facilities ทั่วสหรัฐฯ ใช้ ADS-B + radar ให้บริการทั้ง IFR และ VFR

Facilityบทบาท
TRACON
(Terminal Radar Approach Control)		บริการในเขต terminal ของสนาม Class B/C/TRSA; รวมกับ tower หรือแยกก็ได้ — รับบริการต้องมี transponder code + departure frequency จาก clearance delivery ก่อน taxi; บังคับใช้บริการใน Class B/C
ARTCC
(Air Route Traffic Control Center)		เขต enroute — ให้ flight following; หน้าที่หลักจัดการ IFR แต่ช่วย VFR ตาม workload; พื้นที่กว้างจึงช่วยได้เมื่อหลงทาง
  • สนามนอก Class B/C: ขอ flight following ได้แบบ optional (บริการเตือน traffic) — ขอความถี่จาก tower / Flight Service / airport info
  • หลัง takeoff: tower → departure controller (traffic advisories, safety alerts, vectoring, separation)
  • ขาเข้าสนามที่มี TRACON: ติดต่อ approach control — บริการเหมือน departure + sequencing → hand off ไป tower → clear to land
  • ASDE-X (airport surface detection equipment–model X): รวมข้อมูล radar/ADS-B/transponder เพื่อดู surface movement
Separation ที่ ATC ให้: Class B = แยกจากทั้ง IFR และ VFR · Class C = แยก IFR ออกจาก VFR

8. ATC Services & Pilot Responsibilities

การให้บริการขึ้นกับประเภท facility + workload; controller รับผิดชอบเครื่อง IFR ก่อน — ปริมาณจราจร/ความแออัดของความถี่อาจทำให้ให้บริการ VFR ไม่ได้

บริการคืออะไรความรับผิดชอบของนักบิน
Traffic Advisoryเตือนถึง traffic ที่เกี่ยวข้อง (เช่น "traffic 12 o'clock, 4 miles")นักบินยังรับผิดชอบ collision avoidance — ตอบ "looking for traffic" / "negative contact" / "traffic in sight"
Safety Alertบริการ mandatory ให้ทุกเครื่อง — เตือนเมื่อใกล้ terrain/obstruction/aircraft อันตรายขึ้นกับความสามารถ controller ที่จะเห็นสถานการณ์
Vectorsนำทางแบบ advisory เท่านั้น (controller vector หรือนักบินขอเองได้)flight safety ยังเป็นของนักบิน — แจ้งถ้า vector จะพาเข้า IFR conditions
Sequencingจัดลำดับเครื่องขาเข้าเข้าสู่ pattern อย่างเป็นระเบียบตามเครื่องอื่นได้ แต่ไม่อนุญาตให้ทำตามคำสั่งที่ออกให้เครื่องนั้น; ต้องเลี่ยง wake turbulence เอง
SeparationClass B = แยก IFR+VFR; Class C = แยกจาก IFRต้องขอ clearance เข้า Class B; clearance ไม่ปลดความรับผิดชอบ see-and-avoid, wake turbulence, terrain clearance, รักษา VFR weather

9. การอ่าน Traffic Advisory (ระบบนาฬิกา)

  • ATC อ้างอิงตำแหน่ง traffic เหมือนหน้าปัดนาฬิกา โดย 12 นาฬิกา = หัวเครื่อง
  • เช่น "traffic at 11 o'clock, 2 miles, southbound" = อยู่ 30° ไปทางซ้ายของหัวเครื่อง → มองระหว่างหัวเครื่องกับปีกซ้าย (ช่วง 12 ถึง 9 นาฬิกา)
⚠️ Wind correction angle ไม่แสดงบนจอ ATC — ตำแหน่ง traffic จริงที่เห็นอาจคลาดจากที่ ATC บอกเล็กน้อย (เพราะหัวเครื่องชี้คนละทางกับ track เนื่องจากแก้ลม)

10. Summary Checklist (สรุปจุดสอบ)

  • ADS-B รวม GPS + เครื่องส่ง/รับ + ground stations; Out ส่ง line-of-sight, In รับ traffic แสดงบน CDTI
  • TIS-B = แสดงเครื่องที่มี transponder แต่ไม่มี ADS-B; FIS-B = อากาศ + TFR + SUA
  • Data link: 1090ES ทุกระดับ/บังคับ ≥18,000 ft (ไม่มี FIS-B); 978 UAT <18,000 ft (มี FIS-B)
  • Primary radar = วัดเวลา echo + ทิศ; ATCRBS (secondary) = discrete code แสดง ID/altitude/ground speed
  • ข้อจำกัด radar 3 อย่าง: bending, reflecting, screening
  • Transponder ตั้งได้ 4,096 codes; IDENT = ทำ return blossom; Mode C = altitude; Mode S = data link + บังคับ ADS-B
  • Codes: VFR 1200 · hijack 7500 · comm fail 7600 · emergency 7700
  • squawk = คำที่ controller ใช้สั่ง code/function; discrete code ใน Class B/C หรือเมื่อขอ traffic advisory
  • Transponder inspection ทุก 24 calendar months (controlled airspace)
  • Flight Service: 1-800-WX-BRIEF / 1800wxbrief.com — weather, NOTAMs, file plan; VFR flight plan = SAR protection
  • ไม่ close ภายใน 30 นาทีหลัง ETA → phone search → ถ้าไม่สำเร็จ alert SAR
  • ATIS อัปเดตเมื่อ conditions เปลี่ยน/มี official weather/ต้นชั่วโมง; sky+vis หาย = vis ≥5 ไมล์, ceiling ≥5,000 ft
  • Clearance delivery ออก IFR clearance + อาจให้ VFR departure code; ground control ให้ taxi ใน movement areas
  • TRACON = บริการ terminal (Class B/C/TRSA); ARTCC = flight following ขณะ enroute
  • Traffic advisory: นักบินยังรับผิดชอบ collision avoidance; Separation: Class B (ทุกเครื่อง), Class C (จาก IFR)
  • ATC อ้างอิง traffic แบบนาฬิกา (12 = หัวเครื่อง); wind correction angle ไม่แสดงบนจอ

11. ข้อสอบท้ายบท + เฉลย

คำถามภาษาอังกฤษจากท้ายบท Ch5-A จริง • อ่านคำถาม/ช้อยส์ก่อน แล้วกด "ดูเฉลย" • เฉลยอ้างอิง Appendix A

1.Select the true statement regarding ADS-B system components.

A. ADS-B In transmits line-of-sight signals from aircraft to ATC ground receivers and to aircraft receivers.
B. ADS-B Out transmits line-of-sight signals from aircraft to ATC ground receivers and to aircraft receivers.
C. ADS-B In receives signals from aircraft and displays their lateral position, altitude, and velocity to controllers.

ดูเฉลย

B

2.Select the true statement regarding ADS-B services.

A. Regulations require the CDTI to be a dedicated display.
B. An effective range of 250 nautical miles enhances traffic conflict detection and resolution.
C. ADS-B provides data to ATC that immediately indicates when an aircraft deviates from its assigned flight path.

ดูเฉลย

C

3.Describe three limitations of radar.

ดูเฉลย

The limitations of radar include: bending of radar waves (anomalous propagation), which results in ground clutter; reflection of waves by dense objects, such as heavy clouds, precipitation, ground obstacles, or mountains, which can weaken or eliminate the display of targets at a greater range; and screening by high terrain, which obstructs relatively low-altitude aircraft.

4.Which is true about transponder modes?

A. Mode S provides two-way data link capability and enables ADS-B.
B. Mode A provides 4,096 available codes and transmits your airplane's altitude.
C. Mode C transmits your airplane's altitude and provides two-way data link capability.

ดูเฉลย

A

5.List the standard transponder codes to use for VFR flight, hijacking, communication failure, and emergencies.

ดูเฉลย

VFR — 1200; hijacking — 7500; two-way communication failure — 7600; emergency — 7700

6.True/False. After landing and when clear of the active runway, you should set the transponder to STBY or OFF.

ดูเฉลย

False

7.When should you press IDENT on the transponder and what is the purpose of this function?

ดูเฉลย

ATC asks you to "ident" to cause the transponder return to blossom on the ATC display for a few seconds, enabling the controller to easily identify you.

11.What does it mean if the sky condition and visibility are absent from the ATIS broadcast?

ดูเฉลย

If the sky condition and visibility are absent from the ATIS broadcast, it means that the visibility is 5 miles or more and the cloud ceiling is 5,000 feet or higher.
